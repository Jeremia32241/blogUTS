Menentukan Bilangan Terbesar dari Tiga Bilangan

Deskripsi Masalah:

Algoritma menerima input tiga bilangan untuk menentukan bilangan terbesar dari tiga bilangan tersebut.

Notasi Algoritma Deskriptif:

  1. Mulai
  2. Masukan tiga bilangan bulat (A, B, dan C)
  3. Bandingkan tiga bilangan bulat
  4. Mulai dengan menganggap bilangan A sebagai yang terbesar.
  5. Cek apakah B lebih besar dari A. Jika ya, maka B menjadi yang terbesar.
  6. Selanjutnya, cek apakah C lebih besar dari yang terbesar saat ini. Jika ya, maka C menjadi yang terbesar.
  7. Tampilkan hasil

ini flowchartnya

Ini codenya:

                       
#include <iostream>
using namespace std;
                            
int main() {
    int A, B, C;
cout << "Masukkan tiga bilangan bulat (A, B, C): ";
cin >> A >> B >> C;
                            
int maxAB = A;  
                            
if (B > maxAB) {
    maxAB = B;  
}
if (C > maxAB) {
maxAB = C; 
}
                            
cout << "Bilangan terbesar adalah: " << maxAB << endl;
return 0;
}

Implementasi:

  1. Header File: #include <iostream> digunakan untuk mengizinkan penggunaan fungsi input/output.
  2. Namespace: using namespace std; memungkinkan kita menggunakan elemen dari standar C++ tanpa perlu mengetikkan std:: setiap kali.
  3. Variabel: Mendeklarasikan tiga variabel A, B, dan C untuk menyimpan bilangan yang dimasukkan.
  4. Input: cin >> A >> B >> C; mengambil tiga bilangan dari pengguna.
  5. Inisialisasi: int maxAB = A; menganggap A sebagai bilangan terbesar awal.
  6. Perbandingan: Menggunakan dua pernyataan if untuk membandingkan B dan C dengan maxAB,
  7. memperbarui max AB jika ditemukan bilangan yang lebih besar.
  8. Output: cout digunakan untuk menampilkan bilangan terbesar yang ditemukan.

Ini adalah Pseudocodenya:

Mulai
masukkan A, B, C
maxAB = A
jika B > maxAB, maka maxAB = B
jika C > maxAB, maka maxAB = C
tampilkan maxAB 
Selesai
