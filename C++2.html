Menghitung Faktorial

Deskripsi Masalah:

Algoritma menerima input bilangan yang ingin dihitung faktorialnya

Notasi Algoritma Deskriptif:

  1. Mulai
  2. Masukan bilangan bulat
  3. Jika bukan bilangan bulat tampilkan tidak sesuai
  4. Menghitung factorial (n!=n×(n−1)×(n−2)×…×3×2×1)
  5. Tampilkan hasil

ini flowchartnya

#include  <iostream>
using namespace std;
                            
int main() {
int N;
cout << "Masukkan bilangan bulat non-negatif N: ";
cin >> N;
                            
if (N < 0) {
    cout << "Faktorial tidak terdefinisi untuk bilangan negatif." << endl;
} else {
    long long faktorial = 1;  
                            
for (int i = 1; i <= N; i++) {
    faktorial *= i;  
}
                            
cout << "Faktorial dari " << N << " adalah: " << faktorial << endl;
}
return 0;
}

Implementasi:

  1. Variabel: Mendeklarasikan variabel N untuk menyimpan bilangan yang dimasukkan, dan faktorial untuk menyimpan hasil perhitungan.
  2. Input: Menggunakan cin untuk mengambil nilai N dari pengguna.
  3. Cek Validitas: Pernyataan if memastikan N tidak negatif. Jika N negatif, program memberi tahu bahwa faktorial tidak terdefinisi
  4. Loop untuk Faktorial: Menggunakan for loop untuk menghitung faktorial: Mulai dari 1 hingga N, setiap iterasi mengalikan faktorial dengan i.

Ini adalah Pseudocodenya:

Mulai
masukkan N
jika N < 0 maka
    tampilkan "Faktorial tidak terdefinisi untuk bilangan negatif."
lain
faktorial = 1
untuk i dari 1 hingga N lakukan
    faktorial = faktorial * i
tampilkan "Faktorial dari ", N, " adalah: ", faktorial
Selesai
