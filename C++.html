Menentukan Luas Persegi

Deskripsi Masalah:

Algoritma menerima input sisi lalu menghitungnya menggunakan rumus luas persegi.

Notasi Algoritma Deskriptif:

  1. Mulai
  2. Masukan panjang sisi
  3. Masukan rumus luas persegi (sisi x sisi)
  4. Mulai menghitung luas persegi
  5. Tampilkan hasil

ini flowchartnya

Ini codenya:

#include <iostream>
using namespace std;
                            
int main() {
    double S;
cout << "Masukkan panjang sisi persegi: ";
    cin >> S;
                            
if (S < 0) {
    cout << "Panjang sisi tidak boleh negatif." << endl;
} else {
    double Luas = S * S;  
cout << "Luas persegi adalah: " << Luas << endl;
}
return 0;
}
Code copied to clipboard!

Implementasi:

  1. Variabel: Mendeklarasikan variabel S untuk menyimpan panjang sisi persegi.
  2. Input: Menggunakan cin untuk mengambil nilai panjang sisi dari pengguna
  3. Cek Validitas: Pernyataan if memeriksa apakah S negatif. Jika ya, program memberi tahu pengguna bahwa panjang sisi tidak valid
  4. Hitung Luas: Jika S valid, luas dihitung dengan mengalikan S dengan dirinya sendiri (S * S).
  5. Output: Menampilkan hasil luas menggunakan cout.

Ini adalah Pseudocodenya:

Mulai
Masukkan S Luas = S * S 
Tampilkan Luas
Selesai

Apa itu Flowchart?

Flowchart adalah diagram atau bagan alir yang menggambarkan alur proses, sistem, atau algoritma komputer. Flowchart menggunakan simbol-simbol standar untuk menunjukkan aktivitas, kondisi, dan alur logika dari proses yang digambarkan.
